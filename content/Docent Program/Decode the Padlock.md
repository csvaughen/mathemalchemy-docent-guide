# Activity: Decode the Padlock

The padlock at the center of the [[Cryptography Quilt]] is ringed with **0s and 1s** — a genuine hidden message, not decoration. If you already cracked [[Decode the Banner]], you have every tool you need: it's the same 8-bit ASCII code, hiding a second time in a different scene. *(No spoilers on this page — the message is not printed anywhere on this site.)*

## The hook (docent script)
> "You already learned to read binary at the skateboarder's flag — the quilt's padlock is hiding the exact same kind of message. Want to see if you can crack a second one?"

## How the code works
- **Binary:** every 0 or 1 around the padlock is a **bit**.
- **ASCII:** every 8 bits, read in order, is one character's 8-bit ASCII code — the same rule as the banner. → [[Cryptography]]
- Unlike the banner, exactly how 0 and 1 are shown here (color, shape, or literal digit stitching) hasn't been confirmed yet — look closely at the actual padlock in person to see the convention before grouping bits into bytes.

## Gallery quick hit (3–5 min)
1. Point out the ring of binary around the padlock and remind visitors it's the same code as the banner.
2. Have them find where one 8-bit group starts and ends.
3. Hand them an ASCII decoder card (reuse the banner's) and let them decode the first character — then send them off to find the rest.

## Why this activity earns its spot
The exhibit hides the *same* technique twice, in two different scenes — a deliberate echo, and a great "you already know how to do this" moment for anyone who's already done the banner. → [[Decode the Ring]] is the padlock's neighbor, hiding a second message in a completely different number base.

## Materials
Decoder cards (reuse the banner's ASCII chart — same code). Answer key available from the docent coordinator.
