# Activity: Decode the Padlock

The padlock at the center of the [[Cryptography Quilt]] sits in a field of **0s and 1s** — a genuine hidden message, not decoration. If you already cracked [[Decode the Banner]], you already know the code: it's the same 8-bit ASCII, hidden a second time in a different scene — though the quilt lays it out with a twist of its own. *(No spoilers on this page — the message is not printed anywhere on this site.)*

## The hook 
> "You already learned to read binary at the skateboarder's flag — the quilt's padlock is hiding the exact same kind of message. Want to see if you can crack a second one?"

## How the code works
- **Binary:** every 0 or 1 in the field around the padlock is a **bit**. 
- **ASCII:** every 8 bits is one character's 8-bit ASCII code — the same rule as the banner. → [[Cryptography]]

## Gallery quick hit (3–5 min)
1. Point out the field of binary around the padlock and remind visitors it's the same code as the banner.
2. Have them use the gaps to pick out a single 8-bit letter-group.
3. Hand them an ASCII decoder card — the banner's works for turning a byte into a letter, since the alphabet is identical — and let them decode the first character, then send them off to find the rest.

## Why this activity earns its spot
The exhibit hides the *same* technique twice, in two different scenes — a deliberate echo, and a great "you already know how to do this" moment for anyone who's already done the banner. → [[Decode the Ring]] is the padlock's neighbor, hiding a second message in a completely different number base.

## Materials
No worksheet of its own — the padlock's layout is best read straight off the quilt. The banner's two sheets do the decoding work, since the alphabet is identical:

- **[[Decode the Banner - Decoder Card.pdf|Decoder card (PDF)]]** — turns a group of eight 0s and 1s into a number.
- **[[ASCII Table 0-127.pdf|ASCII table (PDF)]]** — turns that number into a character.

The padlock's layout differs from the banner's, so the *reading order* is worth a fresh look even for visitors who've already done the flag. Answer key available from the docent coordinator.
