# Docent Guide — How to Use

Adapted from the EntreEd Docent Guide (p. 35), which is itself built on the official [Mathematical Connections](https://mathemalchemy.org/mathematical-connections/) pages.

**Activities are still being developed** - there is an index here: [[Activities Index]] and brief descriptions in each Scene.

## The three questions (know these for every scene)
For each of the 16 Connections, a docent should be able to answer:
1. **What math concept is being explained visually?**
2. **How does it relate to you? To your tour group?**
3. **What makes this Connection important — for the present and for the future?**

## The four-beat script (per scene)
Modeled on the guide's [[Cryptography Quilt]] example:

1. **Hook** — "What do you see?" (let the group look and answer first)
2. **Surprise** — "What if I told you that you're looking at ___ right now?"
3. **Explain** — define the one or two key terms in plain language
4. **Engage** — a hands-on moment: assign pairs to a quilt block, run the physical activity, invite a personal connection ("where do you see this in everyday life?")
